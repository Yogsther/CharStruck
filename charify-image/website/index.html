'C'HARSTRUCK 'C'HARSTRUCK is game developed by Olle Kaiser (Yogsther) as an assignment for a game displayed in nothing but words. In the game, you the player need to make it through a few levels. These levels are occupied by monsters that will fight back. The goal is the complete the levels as fast as possible while taking as little damage, and using as few bullets as possible. The game is playable in the web browser and so far has 3 levels.

If you would like to know more about how the game was made, please checkout the repo!
